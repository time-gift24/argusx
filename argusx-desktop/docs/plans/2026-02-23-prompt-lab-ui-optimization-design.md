# PromptLab UI 优化设计方案

**日期**: 2026-02-23
**目标**: 全面优化 PromptLab 界面，增强卡片信息展示和模块关联

## 1. 概述

本次优化旨在提升 PromptLab 的用户体验，主要包括：
- Dashboard 布局优化为九宫格
- 卡片状态信息增强
- 模块之间关联展示

## 2. Dashboard 页面设计

### 2.1 九宫格布局

```
┌─────────────────────────────────────────────────────────┐
│                    PromptLab                            │
├─────────────┬─────────────┬─────────────┬─────────────┤
│  Checklist  │ Golden Sets │   Results   │   Logs      │
│  ─────────  │  ─────────  │  ─────────  │  ─────────  │
│   12 items  │  1 set      │  8 passed   │  24 entries │
│   [查看]    │  [查看]     │  [查看]     │  [查看]     │
├─────────────┴─────────────┴─────────────┴─────────────┤
│                   数据流向可视化                         │
│  ┌──────────→┌──────────→┌──────────→┌──────────┐   │
│  │ Checklist │→ Golden Set│→  Results │→  Logs   │   │
│  └──────────→┌──────────→┌──────────→┌──────────┘   │
│      12          1 set       8/2          24         │
└─────────────────────────────────────────────────────────┘
```

### 2.2 模块卡片结构

每个模块卡片包含：
- **模块图标**: 代表模块的 Lucide 图标
- **模块名称**: 模块标题
- **关键统计**: 主要数字（大字体）
- **快捷操作**: "查看详情" 按钮
- **最近活动**: 可选显示最近活动时间

## 3. 列表页面卡片增强

### 3.1 Checklist 卡片

```
┌────────────────────────────────────────────────────────┐
│  ✓ Check JSON syntax                    [active][step]│
│  ──────────────────────────────────────────────────────│
│  Prompt: Validate that the output is valid JSON...    │
│                                                        │
│  关联: Golden Set #1 | 结果: 5 passed, 1 failed       │
│  创建: 2024-01-15 | 更新: 2024-01-20                  │
│                                                        │
│  [查看] [编辑] [删除]                                  │
└────────────────────────────────────────────────────────┘
```

**新增信息**:
- 关联的 Golden Set 链接
- 检查结果统计（通过/失败数）
- 创建和更新时间

### 3.2 Results 卡片

```
┌────────────────────────────────────────────────────────┐
│  ✓ Passed  │  Context: turn #42                      │
│  ──────────────────────────────────────────────────────│
│  检查项: Check JSON syntax                            │
│  来源: AI | 模型: claude-3-opus                        │
│  耗时: 1.2s | Tokens: 500/200                         │
│  时间: 2024-01-20 14:30:22                             │
│                                                        │
│  [查看详情]                                            │
└────────────────────────────────────────────────────────┘
```

**新增信息**:
- 检查项名称（关联到 Checklist）
- AI 执行详情（模型、token 消耗）
- 执行耗时

### 3.3 Golden Sets 卡片

保持现有结构，增加关联 Checklist 的链接。

## 4. 模块关联展示

### 4.1 Dashboard 底部数据流向图

使用 ASCII 艺术风格展示数据流转：
- Checklist → Golden Set → Results → Logs
- 每个节点显示当前统计数据

### 4.2 卡片内关联

- Checklist 卡片显示关联的 Golden Set
- Results 卡片显示关联的 Checklist 名称（可点击）
- 可点击的关联直接跳转到对应页面

## 5. 技术实现

### 5.1 需要修改的文件

1. `app/prompt-lab/page.tsx` - Dashboard
2. `app/prompt-lab/checklist/page.tsx` - Checklist 列表
3. `app/prompt-lab/golden-sets/page.tsx` - Golden Sets
4. `app/prompt-lab/results/page.tsx` - Results
5. `app/prompt-lab/logs/page.tsx` - Logs（如有）

### 5.2 组件复用

- 提取通用的 StatsCard 组件
- 提取通用的关联信息行组件
